Build a complete modern web application called **Nepal Summer League Finland 2026** for a one-day football (soccer) tournament.

## Purpose

This application will be used during the tournament to:

* Display live fixtures
* Show live scores
* Update standings automatically
* Track match results
* Provide a public link for spectators
* Allow tournament organizers to update scores from a phone

## Teams

1. Kokkola Soccer Boys
2. Jeppis Nepal Klub
3. Oulu Nepalese Sports
4. Seinajoki International Society

## Technical Requirements

* React
* TypeScript
* Tailwind CSS
* Node.js
* Express
