Build a production-ready football tournament management web application called **OSTROBOTHNIA NEPAL SUPER LEAGUE 2026**.

Format: One-Day Football Tournament



Location: Santahaka, Kokkola Finland

Date: 28 June 2026 



Teams:



* Kokkola Soccer Boys

* Jeppis Nepal Klub

* Oulu Nepalese Sports

* Vaasan Nepali Ry



## Goal



Create a modern football tournament web application with live scores, fixtures, standings, statistics, and an admin panel that can be managed entirely from a mobile phone during the event.



The application must be suitable for public spectators, players, and tournament organizers.
